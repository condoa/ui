# Credits

This library adapts code and design patterns from the open-source projects
below, each under the MIT License. Their notices are acknowledged here per the
terms of that license; see each upstream repository for its full license text.

## Component primitives
- reka-ui (https://reka-ui.com): the headless primitive library that many
  components wrap.
- shadcn-vue (https://www.shadcn-vue.com): styling patterns and component
  structure for primitives such as the Sidebar, Dialog, and form controls.

## Animated effects
- Magic UI (https://magicui.design): BorderBeam, Meteors, RetroGrid, and
  NeonGradientCard are adapted from its effect components.
