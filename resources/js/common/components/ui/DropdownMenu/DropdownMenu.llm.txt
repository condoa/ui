# DropdownMenu

Action menu triggered by a button. Use for contextual actions, not selection.

## When to Use
- Row actions in tables
- User menu (profile, settings, logout)
- Context menus
- NOT for form selection (use Select)
- NOT for navigation (use NavigationMenu)

## Usage

Basic:
```vue
<DropdownMenu>
    <DropdownMenuTrigger as-child>
        <Button variant="ghost" size="icon">
            <Icon name="icon-more-horizontal" class="size-4" />
        </Button>
    </DropdownMenuTrigger>
    <DropdownMenuContent>
        <DropdownMenuItem title="Edit" icon="icon-edit" />
        <DropdownMenuItem title="Duplicate" icon="icon-copy" />
        <DropdownMenuSeparator />
        <DropdownMenuItem title="Delete" icon="icon-trash" variant="destructive" />
    </DropdownMenuContent>
</DropdownMenu>
```

With navigation:
```vue
<DropdownMenu>
    <DropdownMenuTrigger as-child>
        <Button variant="outline">Actions</Button>
    </DropdownMenuTrigger>
    <DropdownMenuContent>
        <DropdownMenuItem as-child>
            <Link :href="route('items.edit', item.id)">Edit</Link>
        </DropdownMenuItem>
    </DropdownMenuContent>
</DropdownMenu>
```

## Components

### DropdownMenu
Root component. Manages open state.
- `open` (v-model): controlled open state
- `defaultOpen`: initial open state
- `modal`: trap focus (default: true). When true, pointer events outside the menu are blocked while open.

### DropdownMenuTrigger
Element that opens the menu.
- `asChild`: merge props onto child

### DropdownMenuContent
The dropdown panel.
- `side`: "top" | "right" | "bottom" | "left" (default: "bottom")
- `sideOffset`: offset from trigger (default: 4)
- `align`: "start" | "center" | "end" (default: "center")
- `class`: custom classes

### DropdownMenuItem
Action item.
- `title`: item text
- `subtitle`: secondary text
- `icon`: leading icon
- `shortcut`: keyboard shortcut text
- `variant`: "default" | "destructive"
- `disabled`: disable item
- `inset`: add left padding (for alignment without icon)
- `asChild`: merge props onto child (for links)

### DropdownMenuSeparator
Visual divider.

### DropdownMenuLabel
Non-interactive label.
- `inset`: add left padding

### DropdownMenuGroup
Group items together.

### DropdownMenuCheckboxItem
Toggleable checkbox item.
- `checked` (v-model): checked state

### DropdownMenuRadioGroup / DropdownMenuRadioItem
Radio selection within menu.
```vue
<DropdownMenuRadioGroup v-model="sortBy">
    <DropdownMenuRadioItem value="name">Name</DropdownMenuRadioItem>
    <DropdownMenuRadioItem value="date">Date</DropdownMenuRadioItem>
</DropdownMenuRadioGroup>
```

### DropdownMenuSub / DropdownMenuSubTrigger / DropdownMenuSubContent
Nested submenu.
- `DropdownMenuSubTrigger` props: `title`, `icon`, `inset`, `trailingIcon` (defaults to `icon-chevron-right`)
```vue
<DropdownMenuSub>
    <DropdownMenuSubTrigger title="More options" />
    <DropdownMenuSubContent>
        <DropdownMenuItem title="Option 1" />
        <DropdownMenuItem title="Option 2" />
    </DropdownMenuSubContent>
</DropdownMenuSub>
```

## Accessibility
- Full keyboard navigation
- Arrow keys move between items
- Enter/Space activates item
- Escape closes menu
- Focus returns to trigger on close
- Type-ahead search

## Common Patterns

Table row actions:
```vue
<DropdownMenu>
    <DropdownMenuTrigger as-child>
        <Button variant="ghost" size="icon-sm" class="size-7">
            <Icon name="icon-more-horizontal" class="size-4" />
        </Button>
    </DropdownMenuTrigger>
    <DropdownMenuContent align="end">
        <DropdownMenuItem as-child>
            <Link :href="route('users.edit', user.id)">
                <Icon name="icon-edit" class="size-4" />
                Edit
            </Link>
        </DropdownMenuItem>
        <DropdownMenuSeparator />
        <DropdownMenuItem variant="destructive" @click="deleteUser(user.id)">
            <Icon name="icon-trash" class="size-4" />
            Delete
        </DropdownMenuItem>
    </DropdownMenuContent>
</DropdownMenu>
```

User menu:
```vue
<DropdownMenu>
    <DropdownMenuTrigger as-child>
        <Button variant="ghost" class="gap-2">
            <Avatar :src="user.avatar" :fallback="user.initials" />
            {{ user.name }}
        </Button>
    </DropdownMenuTrigger>
    <DropdownMenuContent class="w-56">
        <DropdownMenuLabel>My Account</DropdownMenuLabel>
        <DropdownMenuSeparator />
        <DropdownMenuItem as-child>
            <Link :href="route('profile')">Profile</Link>
        </DropdownMenuItem>
        <DropdownMenuItem as-child>
            <Link :href="route('settings')">Settings</Link>
        </DropdownMenuItem>
        <DropdownMenuSeparator />
        <DropdownMenuItem @click="logout">Log out</DropdownMenuItem>
    </DropdownMenuContent>
</DropdownMenu>
```

## Don't
- Don't use for form selection (use Select)
- Don't put complex forms in dropdown
- Don't use destructive variant for non-destructive actions

## Related
- Select: for form value selection
- ContextMenu: right-click menu
- Popover: arbitrary floating content
- DataTableRowActions: pre-built row actions
