# Popover

Floating panel with arbitrary content. Use for interactive content that doesn't need a full modal.

## When to Use
- Quick forms (date pickers, color pickers)
- Rich preview content
- Contextual actions with multiple options
- NOT for simple labels (use Tooltip)
- NOT for full forms (use Dialog)
- NOT for action menus (use DropdownMenu)

## Usage

Basic:
```vue
<Popover>
    <PopoverTrigger as-child>
        <Button variant="outline">Open Popover</Button>
    </PopoverTrigger>
    <PopoverContent>
        <div class="space-y-2">
            <h4 class="font-medium">Popover Title</h4>
            <p class="text-sm text-muted-foreground">
                Some content goes here.
            </p>
        </div>
    </PopoverContent>
</Popover>
```

With form:
```vue
<Popover>
    <PopoverTrigger as-child>
        <Button variant="outline">
            <Icon name="icon-calendar" class="size-4 mr-2" />
            Pick date
        </Button>
    </PopoverTrigger>
    <PopoverContent class="w-auto p-0">
        <Calendar v-model="date" />
    </PopoverContent>
</Popover>
```

## Components

### Popover
Root component. Manages open state.
- `open` (v-model): controlled open state
- `defaultOpen`: initial open state
- `modal`: trap focus when open (default: false)

### PopoverTrigger
Element that opens the popover.
- `asChild`: merge props onto child

### PopoverContent
The floating panel.
- `side`: "top" | "right" | "bottom" | "left" (default: "bottom")
- `sideOffset`: offset from trigger (default: 4)
- `align`: "start" | "center" | "end" (default: "center")
- `class`: custom classes

### PopoverArrow
Optional arrow pointing to trigger.

### PopoverClose
Closes the popover when clicked.
- `asChild`: merge props onto child

### PopoverX
Pre-styled close button (absolute top-right). Renders an icon with a screen-reader label.
- `icon`: icon name (default: "heroicons:x-mark")
- `srText`: screen-reader label text (default: "Close")

## Accessibility
- Focus moves into popover when opened
- Escape key closes popover
- Click outside closes popover
- ARIA expanded state on trigger

## Common Patterns

Date picker trigger:
```vue
<Popover>
    <PopoverTrigger as-child>
        <Button variant="outline" class="justify-start text-left font-normal">
            <Icon name="icon-calendar" class="size-4 mr-2" />
            {{ date ? formatDate(date) : 'Pick a date' }}
        </Button>
    </PopoverTrigger>
    <PopoverContent class="w-auto p-0" align="start">
        <VDatePicker v-model="date" />
    </PopoverContent>
</Popover>
```

Quick settings:
```vue
<Popover>
    <PopoverTrigger as-child>
        <Button variant="ghost" size="icon">
            <Icon name="icon-settings" class="size-4" />
        </Button>
    </PopoverTrigger>
    <PopoverContent class="w-80">
        <div class="space-y-4">
            <h4 class="font-medium">Display Settings</h4>
            <div class="flex items-center justify-between">
                <Label for="compact">Compact view</Label>
                <Switch id="compact" v-model="settings.compact" />
            </div>
            <div class="flex items-center justify-between">
                <Label for="density">Density</Label>
                <Select v-model="settings.density">
                    <SelectTrigger class="w-24" />
                    <SelectContent>
                        <SelectItem value="comfortable">Comfortable</SelectItem>
                        <SelectItem value="compact">Compact</SelectItem>
                    </SelectContent>
                </Select>
            </div>
        </div>
    </PopoverContent>
</Popover>
```

User card:
```vue
<Popover>
    <PopoverTrigger as-child>
        <Button variant="link" class="p-0 h-auto">
            {{ user.name }}
        </Button>
    </PopoverTrigger>
    <PopoverContent class="w-64">
        <div class="flex items-center gap-3">
            <Avatar :src="user.avatar" :fallback="user.initials" />
            <div>
                <p class="font-medium">{{ user.name }}</p>
                <p class="text-sm text-muted-foreground">{{ user.email }}</p>
            </div>
        </div>
        <div class="mt-4 flex gap-2">
            <Button size="sm" variant="outline" class="flex-1">Message</Button>
            <Button size="sm" class="flex-1">View Profile</Button>
        </div>
    </PopoverContent>
</Popover>
```

## Don't
- Don't use for simple text (use Tooltip)
- Don't use for complex forms (use Dialog)
- Don't use for action menus (use DropdownMenu)
- Don't make popovers too large

## Related
- Tooltip: simple text labels
- DropdownMenu: action menus
- Dialog: complex forms
- HoverCard: preview on hover (no click)
