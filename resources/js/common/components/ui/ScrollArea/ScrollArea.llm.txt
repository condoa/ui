# ScrollArea

Custom scrollable container with styled scrollbars. Use for consistent cross-browser scroll styling.

## When to Use
- Fixed-height containers with overflow
- Custom scrollbar appearance
- Horizontal scrolling content
- NOT for page-level scroll (use native)

## Usage

Basic vertical scroll:
```vue
<ScrollArea class="h-64">
    <div class="space-y-4 p-4">
        <p v-for="i in 20" :key="i">Item {{ i }}</p>
    </div>
</ScrollArea>
```

Horizontal scroll:
```vue
<ScrollArea orientation="horizontal" class="w-full">
    <div class="flex gap-4 p-4">
        <Card v-for="i in 10" :key="i" class="w-64 shrink-0">
            Card {{ i }}
        </Card>
    </div>
</ScrollArea>
```

## Props

- `orientation`: "vertical" | "horizontal" (default: "vertical")
- `class`: custom classes for root
- `viewportClass`: custom classes for viewport

## Components

### ScrollArea
Root container.

### ScrollAreaViewport
The scrollable content area.

### ScrollAreaScrollbar
The styled scrollbar (rendered automatically).

### ScrollAreaThumb
The scrollbar thumb.

### ScrollAreaCorner
Corner between vertical and horizontal scrollbars.

## Common Patterns

In a dialog:
```vue
<Dialog>
    <DialogContent>
        <DialogHeader>
            <DialogTitle>Select an item</DialogTitle>
        </DialogHeader>
        <ScrollArea class="h-72">
            <div class="space-y-2 p-4">
                <div v-for="item in items" :key="item.id" class="...">
                    {{ item.name }}
                </div>
            </div>
        </ScrollArea>
    </DialogContent>
</Dialog>
```

Dropdown content:
```vue
<SelectContent>
    <ScrollArea class="h-48">
        <SelectItem v-for="item in manyItems" :key="item.value" :value="item.value">
            {{ item.label }}
        </SelectItem>
    </ScrollArea>
</SelectContent>
```

Code preview:
```vue
<ScrollArea class="h-96 rounded-lg border bg-muted">
    <pre class="p-4 text-sm">
        <code>{{ code }}</code>
    </pre>
</ScrollArea>
```

Chat messages:
```vue
<ScrollArea ref="scrollArea" class="h-96" :viewport-class="'flex flex-col-reverse'">
    <div class="space-y-4 p-4">
        <div v-for="message in messages" :key="message.id" class="...">
            {{ message.content }}
        </div>
    </div>
</ScrollArea>
```

## Don't
- Don't use for page-level scrolling
- Don't nest ScrollAreas
- Don't use without explicit height constraint

## Related
- DialogPanel: has built-in scroll
